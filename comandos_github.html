Comandos em github

para abrir o terminal no vscode use o atalho: ctrl + '

comando para mandar o código do vscode para o github com repositorio já criado no github

git init

vincular o código o repositório

git remote add origin link_do_respositorio

adicionar todos os arquivos da pasta para o git

git add *

confirmar as alterações

git commit -m "digite seu comentario"

criar uma branch

git branch -M main

enviar os arquivos do projeto para o repositório

git push -u origin main

ou se for na pasta master bote:

git push -u origin master

para enviar códigos para o github sem criar um local no github

cd nome_do_projeto
git add
git commit -m "comentario sobre o que mudou no projeto"
git push

fazer uma branch/adicionar um nome á branch

git checkout -b nome_da_alteração

depois de fazer alguma alteração no código quero atualizar minha branch no github com o código que fiz no vscode

entrando na branch: git checkout nome_da_branch

aidcionando a aleração do código no vscode para a branch que está no git: git add .

comentario da mudança que foi feita no código: git commit -m "nome_da_alteração"

enviando alterações feita no vscode para a branch do git: git push origin nome_da_branch
no nome_da_branch digite a branch que você quer altualizar com o código novo

quero entrar em alguma branch

git checkout nome_da_branch

quer voltar para a branch principal/main?

git checkout main

quer voltar para a branch que foi criada?

git checkout nome_da_branch em nome_da_branch digite o nome da branch que você deseja entrar

o que é um merge?

merge é juntar uma branch com a branch main/principal

como faço um merge?

primeiro, entre na branch principal: git checkout main
depois bote o nome da branch que você quer juntar/fazer o merge: git merge comentario o comentario necessariamente precisa ser o nome da branch que você quer juntar.
depois envie toda essa alteração para o github: git push origin main

clonando códigos que estão no github

vá até o código que deseja clonar e selecione code depois copie o link do código que deseja clonar e crie uma pasta no seu pc e entre nela pelo vscode, depois digite: git clone cole_o_link_aqui
Em cole_o_link_aqui necessariamente precisa ser o link do repositorio que você deseja clonar

quer pegar uma atualização do código que você clonou?

entre na pasta do repositório clonado usando o comando: cd nome_da_pasta
em seguida use o comando para puxar os arquivos do repositorio do git: git pull
